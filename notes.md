# What Is the Terminal?

- **It's a text-based way to talk to your computer.** Instead of clicking icons, you type commands and press Enter to tell the computer what to do.
- **It's also called the command line or shell.** These words all mean roughly the same thing: a program where you type instructions instead of using a mouse.
- **You navigate folders and files using words, not clicks.** For example, typing `cd Documents` moves you into the Documents folder, like double-clicking it would.
- **It's powerful because it can do things fast and repeatably.** One typed command can do what might take many clicks in a normal app, and you can save commands to reuse later.
- **It looks intimidating but is just a conversation.** You type a command, the computer responds with text, and you keep going — there's no need to memorize everything at once.
- **"Verifying" means double-checking that something actually works the way it's supposed to**, instead of just assuming it does because the code looks right.
